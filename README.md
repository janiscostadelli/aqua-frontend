#  🎧 Aqua
[Aqua](http://ec2-3-83-69-28.compute-1.amazonaws.com:3000/)

Projeto Full Stack de um site de musicas.


#### 📄 Scripts
```
npm start
```
```
npm run build
```

#### ✔️ Features
- Login
- Logout
- Cadastro
- Criar playlist
- Procurar música

#### 📦 API

Clique [aqui](http://aqua-api-doc.surge.sh/) para acessar.
